# File Organizer CLI

A safe, customizable, Python-based file organizer with support for:

- Dry run mode
- Logging (console + file)
- CLI flags for log level and verbosity
- Safe path checks
- Extensible config

## Usage

```bash
python file_organize.py organize ~/Downloads --loglevel DEBUG --logfile
